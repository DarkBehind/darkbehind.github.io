A Lot Of Box






A Lot Of Box Work

This Unity VFX project was developed as a personal growth and self-improvement exercise. The project features a dynamic scene with floating boxes that interact with the viewer's movement. As the viewer passes by the boxes, they shift and move away, creating a sense of depth and motion within the scene. This effect is designed to explore and enhance the user experience in games or interactive media by adding a dynamic and immersive visual element.

Avatar Earthbending Work






Avatar Earthbending Work

In this project, I created an earthbending effect using Unity's Particle System and animation tools. The character stomps the ground, generating a rock particle effect that emerges from the earth. Upon impact with the wall, the rock shatters into smaller pieces, showcasing both the particle system and collision mechanics. This demonstration highlights my ability to create dynamic and visually engaging VFX, combining animation, physics, and particle systems to enhance the realism and impact of character abilities in a game.

VR Waste Recycling Simulation

  • 2022-2023


VR Waste Recycling Simulation

Using Virtual Reality (VR) headsets, this project aims to raise awareness about waste that harms nature and the importance of recycling in a VR environment. The awareness will be conveyed to users online through VR headsets. Users will be able to see each other in the same environment as their digital twins (avatars), communicate in real-time via voice, and complete designated tasks. Thus, the goal is to instill a sense of environmental cleanliness and teach the importance of waste separation in an engaging manner.
